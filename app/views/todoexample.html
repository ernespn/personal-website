Todo with MongoDB

In this example we will be saving simple TODO task into MongoDB.

The backend is done in C# and Java spring. The contract is similar so they should be interchangeable or even load balanced.

The front end made in Angular will do a rest request to those services to get the information about the data stored.

Let's play with it:

There are 3 operations we are interested on; Get all the todos, add new todos and delete todos.

By default every operations we will bring the list of todos, so if nothings come up is because there is nothing in the database.

This is done with GET request.

To insert new todos we do a POST request.

To delete todos we do a DELETE request
