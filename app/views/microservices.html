Microservices with Load Balancing

In this example I have 5 services written in different technologies (Java, C#, F#, Javascript, Python). Each microservices has the same contract are all of them are under the same load balancer.

The angular application is making a call to the loadbalancer, binding to the view and showing the result.

You can see that when a return is given, there is also a from field. That indicates where the response is coming from (c#, java, python, nodejs, f#.)
