Deployment with Docker

This is an example of how to deploy different containers using docker-compose.

In fact. This website was deployed using it.

We have to contanerise all the applications we want. In this case a bunch of microservices, a front end website made in angular (Me!), a load balancer and even a mongoDB database

One that is done, we need link them all toguether using docker-compose file

Then we do a docker-compose up.... and we have them running.

Then we do a docker-compose stop .... and we have them stopped.
